# Blockchain-Demo-app
An demo blockchain app

# Dependencies
* Node Js
* Truffle Ganache
* Mocha - install using ```npm install -g mocha```
* Truffle - install using ```npm install -g truffle```

# Installing and Deploying the app
* open the terminal in the root folder and type
```
   npm install
   truffle compile
   truffle deploy
   ```
# Run the app
 ```npm run blockchain```
 
# Run the test
```mocha```
