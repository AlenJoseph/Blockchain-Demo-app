# Blockchain-Demo-app
An demo blockchain app

# Dependencies
1.Node Js
2.Truffle Ganache
3.Mocha - install using (npm install -g mocha)
4.Truffle - install using (npm install -g truffle)

# Running the app
1.open the terminal in the root folder and type
2.npm install
3.truffle compile
3.truffle deploy
